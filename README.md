# HabiTracker

## Overview
HabiTracker is a habit-tracking web application built using the **MUN stack** (MongoDB, Express.js, Node.js) with **React** for the frontend and **Tailwind CSS** for styling. This project is designed to help users track their daily, weekly, and monthly habits while providing an aesthetically pleasing and functional user interface.

## Features
- **CRUD Operations**:
  - Add, view, update, and delete habits.
- **MongoDB Backend**:
  - All data is stored and managed using MongoDB.
- **Responsive Design**:
  - A user-friendly and visually appealing UI built with React and Tailwind CSS.
- **Local Hosting**:
  - Currently, the application is set up to run locally on your device.

## How to Run Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/aryanpatil97/HabiTracker.git
   ```
2. Navigate to the project directory:
   ```bash
   cd HabiTracker
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the server:
   ```bash
   node -r dotenv/config server.cjs
   ```
5. Run the React app:
   ```bash
   npm start
   ```
6. Access the application at `http://localhost:5001`.
